# **Installation**

To run it in local environment follow these steps:
    
1. Clone/Download the code from this repository

```bash
git clone git@github.com:ISAKids4Ever/RecruitmentApp.git
```

2. Install dependencies 
    
```bash
cd RecruitmentApp
```
